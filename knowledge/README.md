# Portfolio — Knowledge Base
Punto de entrada para cualquier IA o colaborador que trabaje en este portfolio.

## Cómo leer esta documentación
- Si eres una IA: empieza por CONCEPT.md, luego BRAND.md
- Si vas a tocar el código: lee NARRATIVE.md y TYPOGRAPHY.md
- Si vas a escribir copy: lee VOICE.md y COPY.md

## Documentos
- CONCEPT.md — el universo conceptual del portfolio
- BRAND.md — identidad, ventaja injusta y posicionamiento
- NARRATIVE.md — la historia que cuenta el portfolio de inicio a fin
- TYPOGRAPHY.md — sistema tipográfico y por qué
- VOICE.md — cómo suena Ana, qué evita, qué busca
- COPY.md — frases clave, headlines y textos confirmados
